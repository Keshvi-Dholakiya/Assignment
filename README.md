Initialize and Commit:
1. Create a new directory for a project on your local machine.
2. Initialize a Git repository using git init.
3. Create a new file (e.g., README.md) and add some content.
4. Check the status of the repository with git status.
5. Stage the new file using git add.
6. Commit the change with a clear message using git commit -m.
